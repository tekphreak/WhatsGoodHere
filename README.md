# WhatsGoodHere

A crowdsourced Android app that lets users discover and share what's good at any venue — restaurants, bars, events, and more. Find out what to order, what to skip, and what makes a place worth visiting, all from people who've actually been there.

## Download

[📱 Download WhatsGoodHere.apk](./WhatsGoodHere.apk)

> Version 1.0.1 (versionCode 2) — Android

## Features

- 📍 Location-based venue discovery
- 📸 Share photos from venues
- ⭐ Crowdsourced tips and recommendations
- 🗺️ Find what's good near you

## Tech Stack

- React Native + Expo (SDK 55.0.0)
- Kotlin / Gradle 9.0
- Java 17

## Permissions

| Permission | Purpose |
|---|---|
| Location (fine + coarse) | Show nearby venues |
| Camera | Share venue photos |
| Internet | Fetch venue data |
| Audio | In-app recording |

## Package Info

- **Package:** `com.tekphreak.whatsgoodhere`
- **Architectures:** arm64-v8a, armeabi-v7a, x86, x86_64

## Installation

1. Download `WhatsGoodHere.apk` from this repo
2. On your Android device, enable **Install from unknown sources** in Settings
3. Open the APK file to install
